# SauceDemo automation using Selenium, Cucumber and Java

This project is a Selenium, Cucumber, Java and Junit 5 automation framework that tests the Sauce demo shopping website.
It validates that the highest priced product added to the basket.

To run the tests use
_mvn clean test_

* It will launch the browser
* Execute the cucumber scenario
* Generate test reports in the target folder 
* Open cucumber-report.html to see the test report
